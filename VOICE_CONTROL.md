## How to Use

### 1. **Enable Voice Control**

- Click the blue microphone button in the bottom-right corner
- Grant microphone permission when prompted (first time only)
- The button will turn red and pulse when listening

### 2. **Speak Commands**

Speak clearly and naturally. The app will:

- Show what it heard in a toast notification
- Confirm the action with voice feedback
- Execute the command

### 3. **Get Help**

- Click the help icon (?) button to see all available commands
- Or say "Help" while the mic is active

## Available Commands

### Navigation

| Command               | Action                                         |
| --------------------- | ---------------------------------------------- |
| "Next step"           | Move to the next cooking step                  |
| "Previous step"       | Go back to the previous step                   |
| "Go to step [number]" | Jump to a specific step (e.g., "Go to step 3") |
| "Repeat"              | Read the current step again                    |

### Timers

| Command        | Action                               |
| -------------- | ------------------------------------ |
| "Start timer"  | Start the timer for the current step |
| "Pause timer"  | Pause the running timer              |
| "Resume timer" | Resume the paused timer              |

### View Control

| Command            | Action                      |
| ------------------ | --------------------------- |
| "Show ingredients" | Switch to ingredients tab   |
| "Show directions"  | Switch to directions tab    |
| "Read step"        | Read the current step aloud |

### Help

| Command           | Action                      |
| ----------------- | --------------------------- |
| "Help"            | List all available commands |
| "What can I say?" | List all available commands |

## Tips for Best Results

1. **Speak Clearly**: Use a normal speaking voice
2. **Quiet Environment**: Reduce background noise for better recognition
3. **Wait for Feedback**: The app will confirm what it heard
4. **Natural Language**: You can say variations like:
   - "Next" or "Next step" or "Go forward"
   - "Back" or "Previous" or "Go back"
   - "Start the timer" or "Begin timer"

## Privacy & Security

- **No Data Stored**: Voice commands are processed locally in your browser
- **No Cloud Processing**: Uses your browser's built-in speech recognition
- **Microphone Access**: Only active when you click the mic button
- **Automatic Shutoff**: Mic turns off after each command

## Troubleshooting

### Microphone button doesn't appear

- Your browser doesn't support voice control
- Try using Chrome, Edge, or Safari

### "No speech detected" error

- Speak louder or closer to the microphone
- Check your microphone permissions in browser settings
- Ensure your microphone is working (test in system settings)

### Commands not recognized

- Speak more clearly
- Try rephrasing (e.g., "Next" instead of "Next step")
- Check the help modal for exact command phrases
- Reduce background noise

### Voice feedback not working

- Check your device volume
- Ensure browser has permission to play audio
- Try refreshing the page
